## Learning Objectives

By the end of this workshop, participants will be able to:
- Understand how Git tracks versions of files
- Use basic Git commands (`add`, `commit`, `status`, `log`, etc.)
- Work with branches and merge changes
- Resolve merge conflicts
- Push and pull changes between local and remote repositories
- Collaborate with others through GitHub
